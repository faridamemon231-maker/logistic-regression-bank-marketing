# 📊 Logistic Regression Bank Marketing Project

## Overview
This project uses Logistic Regression to predict whether a customer will subscribe to a bank term deposit based on bank marketing campaign data.

## Features
- Data Cleaning & Preprocessing
- Duplicate Removal
- Outlier Detection
- Label Encoding
- Feature Selection using RFE
- Logistic Regression Model
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Results
- Model Accuracy: 89.19%
- Binary Classification Problem
- Effective Feature Selection and Data Preprocessing
